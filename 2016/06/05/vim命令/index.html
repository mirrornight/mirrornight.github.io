插入命令

cw -> 替换单词              (删除光标所在单词)
O -> 在当前行之前插入新行

/pattern -> 搜索模式

复制粘贴命令

p -> 在当前行位置之后粘贴
p -> 在当前行位置之前粘贴
yy -> 复制当前行

undo/redo

u -> undo                   (撤销)
<C-r> -> redo               (恢复)

2016/6/6

打开/保存/退出

:e <path/to/file> -> 打开一个文件
:w -> 保存
:saveas <path/to/file> -> 另存为
ZZ -> 保存并退出
:q! -> 退出不保存
vim 支持打开多个文件可用:bn 和 :bp 来切换下一个或上一个文件

重复

. -> 可以重复上一次的命令
N<command> -> 重复某个命令N次(N为数字)

光标移动

:N -> 到第N行
gg -> 到第一行
G -> 到最后一行
按单词移动:
    1. w -> 到下一个单词的开头
    2. e -> 到下一个单词的结尾
> 如果你认为单词是由默认方式，那么就用小写的e和w。默认上来说，一个单词由字母，数字和下划线组成
> 如果你认为单词是由blank字符分隔符，那么你需要使用大写的E和W
